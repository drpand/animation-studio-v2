# КОНСТИТУЦИЯ СТУДИИ РОДИНА

## 1. ФОРМАТ ПРОЕКТА
- **Платформа:** YouTube
- **Формат:** Аниме-сериал / короткометражки
- **Длительность серии:** Указывается в задаче (по умолчанию 3 минуты ±10 сек)
- **Количество серий:** Указывается в проекте

## 2. ВИЗУАЛЬНЫЙ СТИЛЬ
- **Стиль:** 2.5D аниме
- **Референс:** Сатоси Кон (Satoshi Kon) — сны, реальность, монтаж
- **Цветовая палитра:** Определяется активным проектом (см. project_memory.json)

## 3. ТОН И НАСТРОЕНИЕ
- **Жанр:** Определяется активным проектом
- **Атмосфера:** Кинематографичная, напряжённая, визуальная
- **Запрещено:** Затянутые сцены, нарушение тайминга, галлюцинации персонажей

## 4. ПРАВИЛА ДЛЯ АГЕНТОВ
- Все агенты обязаны учитывать тайминг указанный в задаче.
- Критик обязан проверять соответствие Конституции.
- МЕД-ОТДЕЛ обязан исправлять нарушения Конституции в первую очередь.
- Агенты НЕ должны использовать захардкоженные палитры, имена персонажей или локации из других проектов.
- Контекст проекта загружается из активного файла проекта (project_memory.json).
