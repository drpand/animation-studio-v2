## Примечания

1. **Все polling-циклы** имеют жёсткий лимит: 20 итераций × 15с = 5 мин. После — FAIL с диагностикой.
2. **Проверка AI-контента** — только на структуру: "не пустой", "содержит ключевое слово", "валидный JSON". Никогда — на точный текст, длину > N, конкретные фразы.
3. **Kie.ai** — внешний сервис, тест НЕ должен падать если он недоступен. Проверяем graceful degradation.
4. **БД** — проверяем через API endpoints, не через прямой SQL (это E2E через браузер).
5. **RBAC** — этап 7 исключён, AuthMiddleware закомментирован в `main.py:77`.
